# agenda-citas

Esta es una aplicación web creada con PHP y Mysql para el back-end, y HTML5 CSS con Bootstrap y Javascript para el front-end.

Se ha configurado la creacion de cuentas con contraseña encriptada, inicio de sesion y validacion de usuarios.

Se implementa la navegacion por paginas segun categoria y servicios, se presenta dinamicamente los servicios, tambien las fechas y horas en que no hallan citas previamente programadas para cada empleado.

Se implementa validacion de citas antes de registrarla en la base de datos para evitar duplicidad y se emite alerta en caso de que la cita halla sido tomada por alguien mas en el mismo momento, para que el usuario quede informado y pueda tomar otra hora diferente.

Aun no esta finalizada, esta en fase de desarrollo.
